# CarManager

- Pull all the files from github, open it with visual studio.

- Build dockerfile (you can also pull it from dockerhub)

- Run project buy docker-compose.

- After running, You can view the api doc auto generated by swagger on `localhost:5001/swagger`.

- Once server successfully run, you can start running **CarManagerUI** app.

- The defualt port setting is **5001** for this project, you can change it to suits your need, but do remember to change both of **CarManager**
and **CarManagerUI** app, otherwise it would lost connection to eachother.
